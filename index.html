⬇ Download the Ballsnia pack Prism Launcher instance — auto-updates every launch

Needs: Prism Launcher · Java 21 · ~8 GB RAM free

Install (about 10 minutes, once)

  1. Install Prism Launcher — prismlauncher.org. Sign in with your Microsoft/Minecraft account.
  2. Install Java 21 — adoptium.net (Temurin 21). In Prism: Settings → Java → Auto-detect and pick the Java 21 entry.
  3. Download the pack (button above). Don't unzip it.
  4. Import it — in Prism: Add Instance → Import → choose the ZIP.
  5. RAM is pre-set to 8–10 GB. Only if your PC has 8 GB total, lower it: right-click → Edit → Settings → Memory.
  6. Launch it. The updater runs first — let it finish, then the game opens.
  7. Join — open Multiplayer: Ballsnia is already in your server list. Just select it and click Join Server.

After this, you never re-download — just launch the instance and it auto-updates before the game opens.

Good to know

  • Distant Horizons & shaders are optional/client-side. Toggle them in Edit Instance → Mods and Options → Video Settings → Shader Packs. This pack uses Iris/Sodium — don't install OptiFine, it'll break things.
  • Pokémon on the minimap: Options → Resource Packs → enable Mini's Cobblemon Icons (already included).
If you get a "mod mismatch" / "incompatible mod set" error: close Minecraft, open Prism, launch the pack again and let the updater finish, then rejoin. Still stuck? Delete the instance and re-import the ZIP.
