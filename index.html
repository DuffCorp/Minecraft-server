How to join

Four steps, about 10 minutes — you only do this once.

1

Get Prism Launcher

It's a free, friendly Minecraft launcher. Install it and sign in with your normal Microsoft/Minecraft account.

Get Prism Launcher →
2

Add Java 21

The pack needs Java 21. Install it, then in Prism go to Settings → Java → Auto-detect and pick the Java 21 option.

Get Java 21 →
3

Download & import the pack

Grab the pack with the big yellow button above (don't unzip it). In Prism: Add Instance → Import → choose the ZIP. The icon, mods and server are all built in.

4

Launch & play

Hit Launch and let the updater finish — the game opens on its own. Then open Multiplayer: Ballsnia is already in your list — click it and Join. 🎉

Good to know

  • Pokémon show on your minimap as little icons — already switched on for hunting.
  • Shaders & the huge view distance are optional. Toggle them in Options → Video if your PC feels it. Don't install OptiFine — it breaks the pack.
  • You need to be whitelisted to get in — give Jack your Minecraft username.

Something went wrong?

If you see a "mod mismatch" or "incompatible mod set" message: close Minecraft, open Prism, launch the pack again and let the updater finish, then rejoin. Still stuck? Delete the instance and re-import the ZIP — your worlds and stuff on the server are safe.
